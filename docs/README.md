## Feature Overview

### Core Features

| Feature | Description |
|---------|-------------|
| **Dashboard** | Real-time metrics and alerts |
| **Agents** | Team management and tracking |
| **Clients** | Policyholder management |
| **Production** | Commission and policy tracking |
| **Pending Policies** | Underwriting status |
| **Tasks** | Follow-ups and reminders |
| **Team** | Hierarchy and commission calculator |
| **Settings** | Integration configuration |

### Integrations

| Integration | Data Synced |
|-------------|-------------|
| **MyWFG** | Agents, ranks, licenses, compliance |
| **Transamerica** | Policies, production, pending apps |
| **Gmail** | Automatic OTP retrieval |
