## Production Notes

### Recording Guidelines

When recording these tutorials, follow these best practices:

| Aspect | Recommendation |
|--------|----------------|
| **Resolution** | 1920x1080 (Full HD) minimum |
| **Frame Rate** | 30 fps for screen recordings |
| **Audio** | Clear narration, no background noise |
| **Cursor** | Use highlight effect for visibility |
| **Pacing** | Allow 2-3 seconds between major actions |
| **Branding** | Include WBH logo in intro and outro |

### Editing Checklist

Before publishing each tutorial, verify the following:

1. Audio levels are consistent throughout
2. All on-screen text is readable
3. Transitions are smooth and not jarring
4. No sensitive data (passwords, real client info) is visible
5. Intro and outro are properly branded
6. Video description includes chapter timestamps

### Recommended Tools

| Tool | Purpose |
|------|---------|
| **OBS Studio** | Screen recording (free) |
| **Camtasia** | Screen recording + editing |
| **DaVinci Resolve** | Video editing (free) |
| **Audacity** | Audio editing (free) |
| **Canva** | Thumbnail creation |
