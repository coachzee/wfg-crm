Root Cause Analysis:
--------------------
The production server (crm.wealthbuildershaven.com) is a Manus-hosted web application
running from a checkpoint that was created WITHOUT Chrome/Chromium browsers installed.

The deployed code uses Puppeteer for browser automation (MyWFG login, Transamerica login,
data scraping). Puppeteer requires Chrome to be installed at /root/.cache/puppeteer or
a project-local .chrome-cache directory.

The .chrome-cache directory (609MB) is in .gitignore and not included in the Git repository.
When the Manus checkpoint was created, Chrome was not bundled with it.

The server's start script does `git pull origin main --ff-only` before starting, but
the server has not been restarted since the checkpoint was restored.
